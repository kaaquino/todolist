Todo List em TypeScript

Uso

Basta adicionar um texto e pressionar [enter] para adicionar uma tarefa. O texto pode ser precedido de #1, #2 ou #3 para atribuir prioridades baixa, média ou alta à tarefa. Por padrão, a prioridade será baixa.

Clicar no checkbox da esquerda fará com que a tarefa seja marcada como "feita".

Clicar na lixeira removerá a tarefa da lista.

Ferramentas Utilizadas

  • TypeScript
  • Webpack
  • HTML
  • CSS
